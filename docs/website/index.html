Quick Start Guide

From zero to production in 4 simple steps

1

Install Dependencies

Choose minimal or full installation based on your needs

pip install -r requirements-minimal.txt
2

Train Your Model

Run the training pipeline with your configuration

python pipelines/training_pipeline.py
3

Start API Server

Launch the FastAPI server with auto-generated docs

python -m uvicorn api.main:app --reload
4

Deploy to Production

Container orchestration ready with one command

docker-compose up -d
